# 🚀 WebSSH - 简单好用的远程服务器管理工具

> 就像电脑上的"远程桌面连接"，但是更强大、更漂亮、还能在浏览器里用！

WebSSH 是一个**网页版**的远程服务器管理工具。你可以把它想象成一个"超级遥控器"——不用安装笨重的软件，打开浏览器就能管理你的所有服务器。

---

## 🎯 什么是 WebSSH？—— 用小学生都能懂的话来说

想象你要管理一大堆电脑（服务器），这些电脑可能在天南海北：

- **传统方式**：每台电脑装一个软件，记一堆 IP 地址和密码，打开好多个窗口，手忙脚乱 😫
- **Easy SSH 方式**：打开一个网页，所有服务器都列在上面。点一下就连上了，就像点开一个 APP 一样简单！

|------|---------|---------|

## ✨ 功能详细介绍

### 1. 服务器连接管理 🔗

支持四种协议：
- **SSH** - 最常用的远程连接（Linux 服务器）
- **RDP** - Windows 远程桌面
- **VNC** - 图形化远程桌面
- **Telnet** - 传统远程连接

还可以给服务器**分组**（比如：生产环境、测试环境、开发环境），方便管理。

认证方式两种：
- **密码登录** — 输入密码
- **私钥登录** — 更安全的登录方式（上传私钥文件）

> 🔐 **记住凭据**：勾选后，密码会加密保存在浏览器中，下次连接不用再输入。关闭浏览器自动清除，安全又方便！

### 2. 终端操作 🖥️

连接上服务器后，你会看到一个**黑底白字的命令行窗口**，就和 PuTTY、XShell 一样。

特色功能：
- **快捷键按钮** — 手机和平板上也能轻松操作（ESC、TAB、Ctrl+C 等）
- **最近命令** — 点一下就能重新发送之前用过的命令
- **自适应缩放** — 窗口大小改变时自动适配
- **多会话** — 可以同时连接多台服务器

### 3. SFTP 文件管理 📁

连接服务器后，可以像操作电脑文件管理器一样浏览远程文件：
- 查看文件和文件夹
- 上传文件到服务器
- 从服务器下载文件

### 4. SSH 隧道 🚇

隧道就是给网络数据"挖一条秘密通道"，让你在本地安全地访问远程网络。

三种隧道类型：
- **本地隧道 (-L)** — 把远程的服务映射到本地端口
- **远程隧道 (-R)** — 把本地的服务暴露到远程
- **动态隧道 (-D)** — 创建 SOCKS5 代理，浏览器设置代理后就能安全上网

### 5. Docker 容器管理 🐳

如果你的服务器上装了 Docker，可以直接在网页上：
- 查看所有容器的运行状态
- 启动、停止容器
- 查看容器日志
- 统计信息一目了然（运行中/已停止/镜像数量）

### 6. 命令片段 📝

把常用的命令保存下来，下次直接点击发送到终端，不用重复输入。

支持：
- **变量** — 用 `{var}` 占位，执行时填入具体值
- **标签** — 给片段打标签方便查找（比如：git、docker、nginx）
- **收藏** — 常用片段标记为星标，快速找到
- **导入导出** — 可以和朋友分享你的命令集

### 7. 安全加密 🔐

采用了银行级别的加密标准：

| 技术 | 用途 | 强度 |
|------|------|------|
| **AES-256-GCM** | 加密凭据和备份 | 美国军方标准 |
| **PBKDF2** | 从主密码派生出加密密钥 | 60万次迭代，暴力破解极难 |
| **SHA-256** | 备份完整性校验 | 防篡改 |

简单说就是：**你的密码存在我们这里，比存在银行还安全** 🤯

### 8. 备份管理 💾

备份功能就像游戏的"存档"——不怕丢数据！

- **手动备份** — 随时备份所有服务器配置
- **自动备份** — 设置每天或每周自动备份，还可以限制保留数量
- **加密备份** — 备份可以同时加密保存凭据信息
- **云备份** — 上传到自建的 WebDAV 或 REST 服务器
- **完整性校验** — 每个备份都有 SHA-256 指纹，防止数据被篡改
- **跨机器恢复** — 在另一台电脑上导入备份，所有配置一键恢复

### 9. 主题换肤 🎨

内置四种主题：
- ☀️ **Light** — 浅色主题
- 🌙 **Dark** — 深色主题（护眼）
- 🧛 **Dracula** — 紫色调，程序员最爱
- ❄️ **Nord** — 北欧风格，清新淡雅

### 10. 中英文切换 🌏

在设置面板中可以一键切换界面语言，目前支持：
- 🇨🇳 **简体中文**
- 🇺🇸 **English**

## 🚀 快速开始

### 安装

```bash
# 克隆项目
git clone https://github.com/guoxpeng/webssh.git

# 进入目录
cd webssh

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 使用

1. 打开浏览器访问 `http://localhost:5173`
2. 第一次使用会要求设置**主密码**（用于加密存储的凭据）
3. 点击左侧"服务器"菜单
4. 输入服务器信息（IP、用户名、密码）
5. 点击"连接" → 开始远程管理！
