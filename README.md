## 🚀 怎么安装？

### 方法一：直接下载源码（推荐）

```bash
# 1. 下载代码
git clone https://github.com/guoxpeng/webssh.git

# 2. 进入文件夹
cd webssh

# 3. 安装依赖（就像装软件需要的零件）
npm install

# 4. 启动！（在浏览器打开 http://localhost:5173）
npm run dev
```

### 方法二：配置后端（需要后端配合）

项目默认使用 WebSocket 连接后端的 SSH 代理服务，你需要一个后端服务来桥接 WebSSH 和真实的 SSH 服务器。

## 📖 怎么使用？

### 第一步：设置主密码 🔐

第一次打开网页，会让你设置一个**主密码**。这个密码用来加密你保存的所有服务器密码。

> 就像手机上的锁屏密码，别人拿到你的电脑也看不到你的密码！

### 第二步：添加服务器 🔗

1. 点左边菜单的"服务器"
2. 填上：
   - **名称** — 给服务器起个名字（比如"我的博客服务器"）
   - **主机/IP** — 服务器地址（比如 192.168.1.100）
   - **端口** — 默认 22（不用改）
   - **用户名** — 登录用的账号（比如 root）
   - **密码或私钥** — 登录的钥匙
3. 点击"连接"就开始远程管理了！

### 第三步：开始用吧！🎉

连上之后你会看到一个**黑底白字的窗口**，这就是服务器的命令行。在里面输入命令就能操作远程电脑了。

## 功能速览 📋

### 🖥️ 终端操作
连上服务器后看到的命令行窗口，可以输入各种命令。

### 📁 文件管理
在服务器上浏览、上传、下载文件，像用电脑文件管理器一样。

### 🚇 SSH 隧道
给数据挖一条"秘密通道"，安全访问远程网络。

### 🐳 Docker 管理
如果你的服务器装了 Docker，可以在网页上查看容器状态、启动、停止、看日志。

### 📝 命令片段
把常用的命令存起来，**一键发送**到终端，不用反复输入。

### 💾 备份管理
备份所有服务器的配置，换电脑了**一键恢复**，什么都不丢。

### 🌏 中英文切换
设置面板里点一下，界面就在中文和英文之间切换。

### 🎨 主题换肤
内置 4 种颜色主题：浅色、深色、紫色（Dracula）、北欧风。
