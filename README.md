## 这个工具是干什么的？🤔

你有很多台**远程电脑（服务器）** 要管？

- 以前：装各种软件、记一堆 IP、开好多窗口、手忙脚乱 😫

- ✅ **宏录制 / 自动化**（录制操作→一键回放→批量执行→定时任务）

## 🚀 一键部署

### 方法一：一键安装脚本（推荐）

```bash
curl -fsSL https://raw.githubusercontent.com/guoxpeng/webssh/main/deploy.sh | bash
```

一键完成：下载代码 → 安装依赖 → 构建前端 → 启动服务

然后在浏览器打开 `http://你的服务器IP:9627`

### 方法二：Docker 部署

```bash
docker run -d --restart=unless-stopped --name webssh -p 9627:9627 \
  -e PORT=9627 \
  node:20-alpine sh -c "apk add git openssh && \
  git clone --depth=1 https://github.com/guoxpeng/webssh.git /app && \
  cd /app && npm install && npm run build && node server/index.mjs"
```

### 方法三：手动部署

```bash
# 1. 下载代码
git clone https://github.com/guoxpeng/webssh.git
cd webssh

# 2. 安装依赖
npm install

# 3. 构建前端
npm run build

# 4. 启动服务（单端口运行，前端后端合一）
PORT=9627 node server/index.mjs
```

然后在浏览器打开 `http://localhost:9627`

### 开发模式

```bash
# 终端1：启动后端
node server/index.mjs

# 终端2：启动前端开发服务器（热更新）
npm run dev
```

开发模式前端地址 `http://localhost:5173`，后端 `http://localhost:9627`。

## 📖 怎么使用？

### 第一步：设置主密码 🔐

第一次打开网页，会让你设置一个**主密码**。这个密码用来加密你保存的所有服务器密码。

> 就像手机上的锁屏密码，别人拿到你的电脑也看不到你的密码！

### 第二步：添加服务器 🔗

1. 点左边菜单的"服务器"
2. 填上：
   - **名称** — 给服务器起个名字（比如"我的博客服务器"）
   - **分组** — 归到某个文件夹（比如"生产环境"、"测试环境"）
   - **主机/IP** — 服务器地址（比如 192.168.1.100）
   - **端口** — 默认 22（不用改）
   - **用户名** — 登录用的账号（比如 root）
   - **密码或私钥** — 登录的钥匙
3. 点击"连接"就开始远程管理了！

### 第三步：分组管理 📂

- 点侧栏 `+` 按钮**新建分组**
- 拖拽服务器到不同分组
- 右键分组：**重命名 / 删除 / 一键连接全部**
- 置顶常用服务器到"已置顶"区域
- 分组折叠状态自动保存

### 第四步：宏 / 自动化 🤖

- **录制**：终端工具栏点红点 → 输入命令 → 停止 → 命名保存
- **回放**：在宏面板点 Play，自动逐条发送命令
- **批量执行**：选中宏 → 勾选多台服务器 → 一键全部执行
- **定时任务**：设置宏每小时/每天/每周自动执行

### 第五步：开始用吧！🎉

连上之后你会看到一个**黑底白字的窗口**，这就是服务器的命令行。

**快捷键：**
- `Ctrl+F` — 在终端中搜索
- `Ctrl+K` — 搜索面板
- `Ctrl+P` — 宏面板
- `上下箭头` — 服务器列表键盘导航

## 功能速览 📋

### 🖥️ 终端操作
连上服务器后看到的命令行窗口，可以输入各种命令。
- **终端设置**：齿轮图标调整字体大小、光标样式
- **终端搜索**：`Ctrl+F` 打开搜索栏
- **录制宏**：红点按钮录制操作序列
- **分屏标签**：同时管理多个连接

### 📁 文件管理（SFTP）
在服务器上浏览、上传、下载文件。**双击代码文件直接编辑保存**，不用下载再上传。

### 🤖 宏 / 自动化
- **录制**：红点录制终端操作，自动捕获输入的命令
- **回放**：单机执行或批量在多个服务器上执行
- **定时任务**：按小时/天/周/月自动执行宏，或自定义间隔

### 🚇 SSH 隧道
给数据挖一条"秘密通道"，安全访问远程网络。支持本地（-L）、远程（-R）、动态（-D）三种。

### 🐳 Docker 管理
如果你的服务器装了 Docker，可以在网页上查看容器状态、启动、停止、看日志。

### 📂 Session 分组
服务器列表支持拖拽分组、右键菜单、置顶收藏、折叠状态持久化。

### 📝 命令片段
把常用的命令存起来，**一键发送**到终端，不用反复输入。

### 💾 备份管理
备份所有服务器的配置，换电脑了**一键恢复**。支持自动备份、WebDAV 云同步。

### 📡 串口协议
通过 COM 口连接网络设备（交换机、路由器），支持波特率/数据位/停止位/校验位配置。

### 🌏 中英文切换
设置面板里点一下，界面就在中文和英文之间切换。

### 🎨 主题换肤
内置 4 种颜色主题：浅色、深色、紫色（Dracula）、北欧风。终端独立设置字体和光标。

```
│   ├── macro/           # 宏录制/自动化/批量执行
│   ├── sftp/            # 文件管理 + 内联编辑器
│   ├── snippets/        # 代码片段
│   ├── terminal/        # 终端（展示、设置、分屏）
│   └── tunnel/          # 隧道管理
├── composables/         # 组合式功能（PWA 安装等）
├── layouts/             # 页面布局（侧栏、状态栏）
├── locales/             # 翻译文件（中文/英文）
├── stores/              # 数据管理（Pinia）
├── utils/               # 工具（加密等）
└── views/               # 页面视图
server/
├── index.mjs            # 后端服务（SSH/串口代理 + SFTP + 静态文件）
└── ...
```
